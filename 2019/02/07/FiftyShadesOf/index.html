FiftyShadesOf

[Android Arsenal](https://img.shields.io/badge/Android%20Arsenal-FiftyShadesOf-brightgreen.svg?style=flat)

An elegant context-care loading placeholder for Android

Usage

FiftyShadesOf.with(context)
             .on(view1, view2, view3)
             .start();

[screen](https://raw.githubusercontent.com/florent37/FiftyShadesOf/master/media/fadein.gif)

Available attributes

View Selector

FiftyShadesOf.with(context)

             .on(R.id.view1, R.id.view2, R.id.view3) //views id

             .on(view1, view2, view3) //views references
             
             .on(viewGroup) //group of views
             
             .except(view1, view2) //skip a view

             .start();

Cross

(By Default)

FiftyShadesOf.with(context)
             .on(R.id.view)
             .start();

[screen](https://raw.githubusercontent.com/florent37/FiftyShadesOf/master/media/cross.gif)

Fade

FiftyShadesOf.with(context)
             .on(R.id.view)
             .fadeIn(true)
             .start();

[screen](https://raw.githubusercontent.com/florent37/FiftyShadesOf/master/media/fadein.gif)
